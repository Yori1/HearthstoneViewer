Introduction

I made this portfolio to collect all of my work from my semester of Smart Mobile in a single place.

In the first 4 weeks, I'm attending workshops about Progressive Web Apps and UX design. For every day of Smart Mobile, I'll write a summary of the work I've done and what I took away from the workshops.

Week 1 Day 1
Workshop PWA1

On the first day, I attended the workshop PWA1. I made a simple app with an index.html and the javascript to add the worker and the manifest, to successfully deploy my first PWA-app and install it on my phone.

I was interested in making a more maintainable PWA, and so I looked into combining PWA with Angular.

By using the npm angular/pwa package I managed to add the ServiceWorker module to my Angular application, allowing me to deploy and install my Angular application as a PWA.

Personal research

I decided to start working on my own PWA, I thought of an app that allows the user to view Hearthstone cards, save them lists on their phone, and possibly talk to others about aspects of the cards. I found an API that would allow me to build that, so I looked into caching data from http calls using the Angular ServiceWorker module.

Git for the project: https://github.com/Yori1/hearth-viewer

Week 1 Day 2
Workshop UX 1

I read an article about Lean UX in preparation for the class. In this workshop, we discussed the usability pyramid, and what the foundations of UX are. I made a suggestion for a workshop about iterative design, as I felt that was very relevant to the DUO App assignment.

Portfolio setup

The majority of the day I spent setting up my portfolio.
I decided to make my portfolio in the form of a website, and added it to my Angular app as a separate component.

I looked up how to make the content menu that uses a scrollspy from the Materialize framework.
